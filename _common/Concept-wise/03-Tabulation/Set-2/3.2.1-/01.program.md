## Title
Flag Route Tabulator

## Concepts Involved
Tabulation, State Tracking, Counting DP

## Problem Statement
A volunteer is decorating a festival staircase with flags.

The staircase has `n` steps, numbered from `1` to `n`.
In one move, the volunteer may climb exactly:
- `1` step,
- `2` steps, or
- `3` steps.

However, the volunteer does not want the decoration pattern to look repetitive.
So the same jump size cannot be used in two consecutive moves.

Count the total number of valid jump sequences that reach exactly step `n`.

A sequence is valid if:
- the sum of all jump sizes is exactly `n`, and
- no two adjacent jumps have the same size.

Two sequences are considered different if at least one jump size differs at some position.
Print the exact count of valid sequences.

## Input Format
- A single integer `n`

## Output Format
- Print the total number of valid jump sequences that reach exactly step `n`

## Constraints
- `1 <= n <= 5000`

## Examples
### Example 1
Input:
`1`

Output:
`1`

### Example 2
Input:
`3`

Output:
`3`

### Example 3
Input:
`4`

Output:
`3`

### Example 4
Input:
`5`

Output:
`5`

## Edge-Case Thinking Prompts
- What happens when `n = 1`?
- Can the sequence `1, 1` be counted?
- Why is a direct jump of `3` valid for `n = 3`?
- Is knowing only the current step enough?
- Why do we need to remember the last jump size?

## Thinking Like a Programmer
- What are the choices at each move?
- What information affects the next choice?
- How can we split answers based on the last jump used?
- How do smaller step counts build larger step counts?
- Where should the final answer be stored?
