## Title
Beacon Iterative Growth

## Concepts Involved
Tabulation, Recurrence Execution, 1D DP

## Problem Statement
A lighthouse beacon stores its brightness level for every day.

For the first two days, the brightness values are fixed:
- Day `1` brightness = `1`
- Day `2` brightness = `2`

From day `3` onward, the brightness for the current day is normally calculated as:

`brightness[i] = brightness[i - 1] + brightness[i - 2]`

However, the beacon has a maintenance drain.
Whenever the day number is a multiple of `4`, the computed brightness for that day is reduced by `1`.

Find the brightness value on day `n` using an iterative bottom-up DP approach.

Print the exact brightness value.

## Input Format
- A single integer `n`

## Output Format
- Print the brightness value on day `n`

## Constraints
- `1 <= n <= 5000`

## Examples
### Example 1
Input:
`1`

Output:
`1`

### Example 2
Input:
`4`

Output:
`4`

### Example 3
Input:
`8`

Output:
`28`
