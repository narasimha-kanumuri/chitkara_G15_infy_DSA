## Plain-English Implementation Walkthrough
1. We need to cross beyond the last real step.  
   So the code creates a virtual finish position `n + 1`.

2. The virtual finish has no toll.  
   So `cost[n + 1]` remains `0`.

3. We must remember whether the coupon is unused or used.  
   So the code creates two DP arrays.

4. `withoutCoupon[0] = 0` means the agent starts on the ground with no toll paid.  
   The coupon is still unused at this point.

5. `withCoupon[0] = INF` because the coupon cannot already be used before the journey starts.  
   This prevents invalid transitions.

6. Every position can be reached from `position - 1` or `position - 2`.  
   So the code checks both previous positions when possible.

7. To keep the coupon unused at the current position, the code adds `cost[position]`.  
   This means the current toll is paid normally.

8. To reach the used-coupon state, the code compares two choices.  
   The coupon may have been used earlier, or it may be used on the current real step.

9. The code allows coupon use only when `position <= n`.  
   This blocks coupon use on the virtual finish.

10. At the end, both final states are valid.  
    So the code returns the smaller final cost.

## Complete Solution Code
```java
import java.io.BufferedInputStream;
import java.io.IOException;
import java.util.Arrays;

public class Main {
    private static final long INF = Long.MAX_VALUE / 4;

    private static long minimumToll(int n, long[] cost) {
        long[] withoutCoupon = new long[n + 2];
        long[] withCoupon = new long[n + 2];

        Arrays.fill(withoutCoupon, INF);
        Arrays.fill(withCoupon, INF);

        withoutCoupon[0] = 0;

        for (int position = 1; position <= n + 1; position++) {
            long bestWithoutCoupon = withoutCoupon[position - 1];
            long bestWithCoupon = withCoupon[position - 1];

            if (position >= 2) {
                bestWithoutCoupon = Math.min(bestWithoutCoupon, withoutCoupon[position - 2]);
                bestWithCoupon = Math.min(bestWithCoupon, withCoupon[position - 2]);
            }

            withoutCoupon[position] = bestWithoutCoupon + cost[position];

            long couponUsedEarlier = bestWithCoupon + cost[position];
            long couponUsedHere = INF;

            if (position <= n) {
                couponUsedHere = bestWithoutCoupon;
            }

            withCoupon[position] = Math.min(couponUsedEarlier, couponUsedHere);
        }

        return Math.min(withoutCoupon[n + 1], withCoupon[n + 1]);
    }

    public static void main(String[] args) throws Exception {
        FastScanner fs = new FastScanner();
        int n = fs.nextInt();

        long[] cost = new long[n + 2];
        for (int i = 1; i <= n; i++) {
            cost[i] = fs.nextLong();
        }

        System.out.println(minimumToll(n, cost));
    }

    private static class FastScanner {
        private final BufferedInputStream in = new BufferedInputStream(System.in);
        private final byte[] buffer = new byte[1 << 16];
        private int ptr = 0;
        private int len = 0;

        private int read() throws IOException {
            if (ptr >= len) {
                len = in.read(buffer);
                ptr = 0;
                if (len <= 0) {
                    return -1;
                }
            }
            return buffer[ptr++];
        }

        int nextInt() throws IOException {
            return (int) nextLong();
        }

        long nextLong() throws IOException {
            int c;
            do {
                c = read();
            } while (c <= ' ');

            long sign = 1;
            if (c == '-') {
                sign = -1;
                c = read();
            }

            long value = 0;
            while (c > ' ') {
                value = value * 10 + (c - '0');
                c = read();
            }

            return value * sign;
        }
    }
}
```

## Code Deconstruction
- `minimumToll(int n, long[] cost)`
  - Computes the minimum toll to reach the virtual finish.
- `withoutCoupon[position]`
  - Best cost to reach `position` with coupon unused.
- `withCoupon[position]`
  - Best cost to reach `position` after coupon has been used.
- `withoutCoupon[0] = 0`
  - The journey starts at the ground with zero toll.
- `bestWithoutCoupon` and `bestWithCoupon`
  - Store the best previous costs from `position - 1` and `position - 2`.
- `couponUsedEarlier`
  - Continues a path where the coupon was already used before this position.
- `couponUsedHere`
  - Uses the coupon on the current real step, so the current toll is not added.
- `position <= n`
  - Ensures coupon is used only on a real toll step.
- Final return
  - Takes the smaller value at the virtual finish because coupon use is optional.

## Language-Specific Notes
- Use `long`, not `int`, because toll totals may be large.
- `INF = Long.MAX_VALUE / 4` avoids overflow during addition.
- `Arrays.fill` is used because almost all DP cells start as unreachable.
- The virtual finish simplifies the final answer.
- The solution is iterative, so it avoids recursion depth issues.
