# Cached Toll Climb

## Concepts Involved

Memoization, State Design, Optimization DP, Recursion to DP, Coupon State

## Problem Statement

A courier has to cross a staircase with `n` toll steps.

The toll amount of step `i` is given in `cost[i]` using `0`-based indexing.
The courier starts before the first step, but the first move must land on step `0`.

After landing on a step, the courier may move either `1` step or `2` steps forward.
If the courier moves beyond the last step, the climb is complete.

Each landed step normally adds its toll cost to the total cost.
The courier has one coupon that can cancel the toll of exactly one landed step.
The coupon may be used at most once.

Find the minimum total toll needed to finish the climb.

## Input Format

- The first line contains an integer `n`.
- The second line contains `n` integers, where `cost[i]` is the toll of step `i`.

## Output Format

- Print the minimum total toll needed to move beyond the last step.

## Constraints

- `1 <= n <= 2000`
- `1 <= cost[i] <= 10^9`
- The answer fits in a signed 64-bit integer.

## Examples

### Example 1

Input:

```text
5
10 15 20 5 10
```

Output:

```text
15
```

Explanation:

One optimal route is:

```text
step 0 -> step 1 -> step 3 -> finish
```

The normal cost is:

```text
10 + 15 + 5 = 30
```

Use the coupon on step `1`, so the paid cost becomes:

```text
10 + 0 + 5 = 15
```

### Example 2

Input:

```text
3
5 100 5
```

Output:

```text
5
```

Explanation:

The courier starts by landing on step `0`.
Then either of these routes can be optimal:

```text
step 0 -> step 2 -> finish
step 0 -> step 1 -> finish
```

Using the coupon on one expensive or required landed step gives minimum cost `5`.

### Example 3

Input:

```text
1
8
```

Output:

```text
0
```

Explanation:

The courier must land on step `0`.
The coupon can cancel that toll completely.

## Edge-Case Thinking Prompts

- What happens when there is only one step?
- Should the coupon be part of the recursion state?
- If two paths reach the same step, are they always equivalent?
- Why does the answer change depending on whether the coupon was already used?
- When the courier moves beyond the last step, should any extra cost be added?
