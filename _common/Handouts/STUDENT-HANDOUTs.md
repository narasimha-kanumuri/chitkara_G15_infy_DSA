## ✅ DAY-WISE HANDOUT

---

## 📘 DAY 1 — Recursion Thinking

- **🎯 You will learn:**
- Base case
- Recursive calls
- How problems shrink

- **🧠 Pattern:**
- 👉 “Solve smaller version of same problem”

- **🧩 Homework:**
- **Easy:**
- Basic Fibonacci recursion
- **Medium:**
- Climbing stairs (recursive)
- **Stretch:**
- Frog jump (recurrence only)

- **✅ Outcome:**
- Understand recursion tree
- Identify base case properly

---

## 📘 DAY 2 — Memoization (DP Bridge)

- **🎯 You will learn:**
- Overlapping subproblems
- Cache results

- **🧠 Pattern:**
- 👉 “Same problem repeating → store result”

- **🧩 Homework:**
- **Easy:**
- Fibonacci with memo
- **Medium:**
- Min cost climbing stairs
- **Stretch:**
- Subset/backtracking hybrid

- **✅ Outcome:**
- Convert recursion → memo

---

## 📘 DAY 3 — 1D Linear DP ✅ (IMPORTANT)

- **🎯 You will learn:**
- dp[i] meaning
- Transition logic
- Tabulation

- **🧠 Pattern:**
- 👉 “Answer depends on previous index”

- **🧩 Homework:**
- **Easy:**
- Count ways climb stairs
- **Medium:**
- Minimum cost stairs
- **Stretch:**
- Frog jump with k steps

- **✅ Outcome:**
- Write dp[i] correctly
- Build forward logic
