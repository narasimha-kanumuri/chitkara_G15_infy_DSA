## Program.md

### Title
Security Bit Schedule

### Concepts Involved
Dynamic Programming, Binary-State DP, Counting DP, Constraint Handling

### Problem Statement
A control room must generate a binary security schedule of length `n`.
Each position in the schedule contains either:
- `0` meaning normal monitoring
- `1` meaning alert monitoring

The schedule must follow these rules:
- It must not contain two consecutive `1`s.
- It must end with `0`.
- Some positions are maintenance slots where `1` is not allowed.

Count the total number of valid security schedules.
Since the answer can be large, return it modulo `1_000_000_007`.

### Input Format
- Integer `n` — length of the schedule
- Integer `m` — number of maintenance slots
- `m` integers representing maintenance positions where `1` is not allowed

Positions are 1-indexed.

### Output Format
- Print the number of valid schedules modulo `1_000_000_007`

### Constraints
- `1 <= n <= 100000`
- `0 <= m <= n`
- `1 <= maintenancePosition <= n`
- Maintenance positions are distinct

### Examples

#### Example 1
Input:
```text
4 1
2
```

Output:
```text
4
```

Explanation:
Position `2` cannot contain `1`.
The valid schedules ending with `0` are:
- `0000`
- `0010`
- `1000`
- `1010`

Total = `4`.

#### Example 2
Input:
```text
5 0
```

Output:
```text
8
```

Explanation:
With no maintenance slots, valid schedules ending with `0` are counted normally under the no-consecutive-`1`s rule.

#### Example 3
Input:
```text
1 1
1
```

Output:
```text
1
```

Explanation:
The only valid ending schedule is `0`.
Maintenance blocks `1`, but `0` is still allowed.

### Edge-Case Thinking Prompts
- What if `n = 1`?
- What if all positions are maintenance positions?
- Why does the answer require only schedules ending with `0`?
- Why is knowing only the current position not enough?
- What changes when the previous bit is `1`?

### Thinking Like a Programmer
- At each position, should we place `0` or `1`?
- If we place `1`, what must the previous bit be?
- If the current position is a maintenance slot, which choice becomes forbidden?
- Can we count schedules based on how they end?
