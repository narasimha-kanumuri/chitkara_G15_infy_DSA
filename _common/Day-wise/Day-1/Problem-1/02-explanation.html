The Mix-Up

Badges: 1, 2, 2, 4, 5

Find the duplicate and the missing!

Deconstructing the Problem

  • Roster: 1 to n
  • Input: list of n numbers
  • Glitch: one duplicate, one missing
  • Output: [Duplicate] [Missing]

The Human Approach

Clipboard tally marks for each badge.

  • Badge 1 → Slot 1
  • Badge 2 → Slot 2
  • Badge 2 again → Slot 2 (Duplicate!)
  • Badge 4 → Slot 4
  • Badge 5 → Slot 5
  • Slot 3 empty → Missing!

Clipboard → Computer

Clipboard = Frequency Array

Original: [1,2,2,4,5]

Tally Array: [0,0,0,0,0,0]

Step-by-Step Dry Run

StepBadgeTally SlotArray
111[0,1,0,0,0,0]
222[0,1,1,0,0,0]
322[0,1,2,0,0,0]
444[0,1,2,0,1,0]
555[0,1,2,0,1,1]

Gathering the Report

  • Slot 2 → Count 2 → Duplicate
  • Slot 3 → Count 0 → Missing

Output: 2 3

Algorithm Logic


1. Create tally[n+1] = 0
2. For each badge:
     tally[badge]++
3. Scan tally:
     if count==2 → duplicate
     if count==0 → missing
4. Print duplicate, missing

Reality Check

  • Time: O(N) → Fast
  • Space: O(N) → Needs extra array

Perfect for beginners!
