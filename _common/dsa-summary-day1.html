Sorting Algorithms

Quick Sort

Divide & conquer, average O(n log n)

Merge Sort

Stable, O(n log n), great for linked lists

Bubble Sort

Simple but inefficient, O(n²)

Problem-Solving Framework

  1. Understand the problem
  2. Choose the right DS
  3. Pick/Design algorithm
  4. Analyze complexity
  5. Test edge cases

Complexity Cheatsheet

Big-O Notation Quick Recall:

  • O(1) → Constant
  • O(log n) → Logarithmic
  • O(n) → Linear
  • O(n log n) → Efficient sorting
  • O(n²) → Nested loops
