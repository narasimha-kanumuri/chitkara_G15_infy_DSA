## Complete Solution Code

```java
import java.io.*;
import java.util.*;

public class Solution {
    public static long minimumEnergy(int n, long[] height, long penalty) {
        if (n == 1) return 0;

        long INF = Long.MAX_VALUE / 4;
        long[] endWithOne = new long[n + 1];
        long[] endWithTwo = new long[n + 1];

        Arrays.fill(endWithOne, INF);
        Arrays.fill(endWithTwo, INF);

        endWithOne[1] = 0;
        endWithTwo[1] = 0;

        if (n >= 2) {
            endWithOne[2] = Math.abs(height[2] - height[1]);
        }

        for (int i = 3; i <= n; i++) {
            long oneStepCost = Math.abs(height[i] - height[i - 1]);
            long minPrevForOne = Math.min(endWithOne[i - 1], endWithTwo[i - 1]);
            if (minPrevForOne != INF) {
                endWithOne[i] = minPrevForOne + oneStepCost;
            }

            long twoStepCost = Math.abs(height[i] - height[i - 2]);
            
            if (endWithOne[i - 2] != INF) {
                endWithTwo[i] = Math.min(endWithTwo[i], endWithOne[i - 2] + twoStepCost);
            }
            
            if (endWithTwo[i - 2] != INF) {
                endWithTwo[i] = Math.min(endWithTwo[i], endWithTwo[i - 2] + twoStepCost + penalty);
            }
        }

        return Math.min(endWithOne[n], endWithTwo[n]);
    }

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());
        if (!st.hasMoreTokens()) return;
        
        int n = Integer.parseInt(st.nextToken());
        long[] height = new long[n + 1];
        
        st = new StringTokenizer(br.readLine());
        for (int i = 1; i <= n; i++) {
            height[i] = Long.parseLong(st.nextToken());
        }
        
        st = new StringTokenizer(br.readLine());
        long penalty = Long.parseLong(st.nextToken());
        
        System.out.println(minimumEnergy(n, height, penalty));
    }
}
```

## Code Deconstruction

- **minimumEnergy(int n, long[] height, long penalty):**
- Computes the minimum total energy to reach stone n.
- **endWithOne[i]:**
- Minimum energy to reach stone i when the last jump was length 1.
- **endWithTwo[i]:**
- Minimum energy to reach stone i when the last jump was length 2.
- **endWithOne[1] = 0 and endWithTwo[1] = 0:**
- The frog starts at stone 1.
- Both states are set to 0 so the first jump does not incorrectly receive fatigue.
- **oneStepCost:**
- Cost of jumping from i - 1 to i.
- A 1-jump can follow either previous state.
- **twoStepCost:**
- Cost of jumping from i - 2 to i.
- **fromPreviousOne:**
- A 2-jump after a 1-jump.
- No fatigue penalty is added.
- **fromPreviousTwo:**
- A 2-jump after another 2-jump.
- Fatigue penalty is added.
- **Final result: Math.min(endWithOne[n], endWithTwo[n]):**
- The path may end with either a 1-jump or a 2-jump.

## Built-ins / Syntax Reference Table

| Java syntax / built-in | Meaning / Role |
| :--- | :--- |
| Arrays.fill(arr, INF) | initializes all DP cells as unreachable |
| Math.abs(a - b) | computes jump energy from height difference |
| Math.min(a, b) | selects cheaper transition |
| long | safely stores large energy totals |
| BufferedInputStream | reads large input efficiently |

## Language-Specific Notes

- long is required because both height differences and accumulated energy may be large.
- INF = Long.MAX_VALUE / 4 avoids overflow while adding costs.
- The iterative DP avoids recursion depth limits.
- The fast scanner is used because n can be large.
- This implementation is 1-indexed to match the problem statement.
