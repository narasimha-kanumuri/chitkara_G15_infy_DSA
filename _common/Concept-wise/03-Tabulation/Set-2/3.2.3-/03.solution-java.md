## Plain-English Implementation Walkthrough
1. We need to minimize total energy.  
   So every DP cell stores the best known energy for one state.

2. Stone number alone is not enough.  
   So the code stores three dimensions: stone, last jump was `2`, and special jump used.

3. Most states are not reachable at the start.  
   So the code fills all DP cells with `INF`.

4. The robot starts at stone `1`.  
   There is no previous `2`-jump, and the special jump is unused.

5. From each reachable state, the code tries a `1`-stone jump.  
   This makes `lastTwo` become `0` because the latest jump is not `2`.

6. The code also tries a `2`-stone jump.  
   If the previous state already had `lastTwo = 1`, it adds the fatigue penalty.

7. The code tries a `3`-stone jump only when `usedThree = 0`.  
   This enforces the at-most-once special jump rule.

8. A `3`-stone jump uses double height-difference cost.  
   So the code multiplies that jump cost by `2`.

9. A `3`-stone jump does not count as a `2`-stone jump.  
   So after using it, `lastTwo` becomes `0`.

10. At the last stone, any state is acceptable.  
    So the code takes the minimum over all four final states.

## Complete Solution Code
```java
import java.io.BufferedInputStream;
import java.io.IOException;
import java.util.Arrays;

public class Main {
    private static final long INF = Long.MAX_VALUE / 4;

    private static long minimumEnergy(int n, long[] height, long penalty) {
        if (n == 1) {
            return 0;
        }

        long[][][] dp = new long[n + 1][2][2];

        for (int i = 0; i <= n; i++) {
            for (int lastTwo = 0; lastTwo <= 1; lastTwo++) {
                Arrays.fill(dp[i][lastTwo], INF);
            }
        }

        dp[1][0][0] = 0;

        for (int stone = 1; stone <= n; stone++) {
            for (int lastTwo = 0; lastTwo <= 1; lastTwo++) {
                for (int usedThree = 0; usedThree <= 1; usedThree++) {
                    long current = dp[stone][lastTwo][usedThree];
                    if (current >= INF) {
                        continue;
                    }

                    if (stone + 1 <= n) {
                        long cost = Math.abs(height[stone + 1] - height[stone]);
                        dp[stone + 1][0][usedThree] = Math.min(dp[stone + 1][0][usedThree], current + cost);
                    }

                    if (stone + 2 <= n) {
                        long cost = Math.abs(height[stone + 2] - height[stone]);
                        if (lastTwo == 1) {
                            cost += penalty;
                        }
                        dp[stone + 2][1][usedThree] = Math.min(dp[stone + 2][1][usedThree], current + cost);
                    }

                    if (usedThree == 0 && stone + 3 <= n) {
                        long cost = 2L * Math.abs(height[stone + 3] - height[stone]);
                        dp[stone + 3][0][1] = Math.min(dp[stone + 3][0][1], current + cost);
                    }
                }
            }
        }

        long answer = INF;
        for (int lastTwo = 0; lastTwo <= 1; lastTwo++) {
            for (int usedThree = 0; usedThree <= 1; usedThree++) {
                answer = Math.min(answer, dp[n][lastTwo][usedThree]);
            }
        }

        return answer;
    }

    public static void main(String[] args) throws Exception {
        FastScanner fs = new FastScanner();

        int n = fs.nextInt();
        long[] height = new long[n + 1];

        for (int i = 1; i <= n; i++) {
            height[i] = fs.nextLong();
        }

        long penalty = fs.nextLong();
        System.out.println(minimumEnergy(n, height, penalty));
    }

    private static class FastScanner {
        private final BufferedInputStream in = new BufferedInputStream(System.in);
        private final byte[] buffer = new byte[1 << 16];
        private int ptr = 0;
        private int len = 0;

        private int read() throws IOException {
            if (ptr >= len) {
                len = in.read(buffer);
                ptr = 0;
                if (len <= 0) {
                    return -1;
                }
            }
            return buffer[ptr++];
        }

        int nextInt() throws IOException {
            return (int) nextLong();
        }

        long nextLong() throws IOException {
            int c;
            do {
                c = read();
            } while (c <= ' ');

            long sign = 1;
            if (c == '-') {
                sign = -1;
                c = read();
            }

            long value = 0;
            while (c > ' ') {
                value = value * 10 + (c - '0');
                c = read();
            }

            return value * sign;
        }
    }
}
```

## Code Deconstruction
- `minimumEnergy(int n, long[] height, long penalty)`
  - Computes the minimum energy needed to reach stone `n`.
- `dp[stone][lastTwo][usedThree]`
  - Stores the best energy for one exact state.
- `lastTwo`
  - `0` means the previous jump was not `2`.
  - `1` means the previous jump was `2`.
- `usedThree`
  - `0` means the special `3`-stone jump is still unused.
  - `1` means it has already been used.
- `dp[1][0][0] = 0`
  - The robot starts on the first stone with zero energy.
- `stone + 1`, `stone + 2`, and `stone + 3` checks
  - These prevent jumps beyond the final stone.
- `lastTwo == 1`
  - This condition decides whether fatigue penalty is added to a `2`-stone jump.
- `usedThree == 0`
  - This condition allows the special jump only once.
- Final answer loop
  - The final state may have any last-jump and special-jump status, so all are checked.

## Language-Specific Notes
- `long` is used to avoid overflow for large height differences and accumulated energy.
- `INF = Long.MAX_VALUE / 4` helps avoid overflow when adding costs.
- A three-dimensional array is readable here because there are only four state combinations per stone.
- `Arrays.fill` is used on the innermost arrays.
- The solution is iterative, so it avoids recursion depth problems.

- Time Complexity: `O(n)`
